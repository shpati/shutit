# shutit
Shutdown program for Windows that shows countdown and allows cancelling the command. Written in C. MIT license.

Default use with no arguments given, shuts down in 60s:
```shutit``` 

Shuts down in 10s:
```shutit /s 10```

Restarts in 20s:
```shutit /r 20```

Hipernates in 30s:
```shutit /h 30```

Sleeps in 60s:
```shutit /sleep 60```
